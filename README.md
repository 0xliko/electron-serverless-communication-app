## Electron app.
### Basic setup with Typescript and Webpack.

To run this project.

- Clone this repo

```bash
  git clone https://github.com/jsgrowing315/electron-serverless-communication-app.git
```

- Install the dependencies

```bash
  npm install
```

- run this commands in  console.


```bash
  npm run dev:electron
